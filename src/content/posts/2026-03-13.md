## 论文详解

### 1. IsoCompute Playbook: Optimally Scaling Sampling Compute for LLM RL
**作者**: Zhoujun Cheng, Yutao Xie, Yuxiao Qu, Amrith Setlur 等 (CMU, MBZUAI, UCSD)  
**链接**: [arXiv:2603.12151](https://arxiv.org/abs/2603.12151)  
**方向**: LLM RL / Scaling Laws

**核心创新**:
预训练有 scaling laws 指导，但 LLM 的 RL 后训练中 sampling compute 的最优分配仍缺乏系统性研究。本文将 RL scaling 建模为三个资源的约束优化问题：每问题的并行 rollout 数、每 batch 的问题数、更新步数。

研究发现：
- **并行 rollout 数**随算力预算增加而增加，但会饱和
- **简单问题**：增加 rollout 主要通过 "solution sharpening" 提升性能
- **困难问题**：增加 rollout 主要通过 "coverage expansion" 扩展解空间
- **问题间干扰**：增加并行 rollout 可缓解不同问题间的干扰
- **训练稳定性**：每 batch 的问题数主要影响稳定性，可在较宽范围内选择

**实验结果**:
在多个 base model 和数据分布上验证，为 compute-efficient 的 LLM RL 后训练提供了实用的算力分配规则。

### 2. IndexCache: Accelerating Sparse Attention via Cross-Layer Index Reuse
**作者**: Yushi Bai 等 (清华大学, 智谱AI)  
**链接**: [arXiv:2603.12201](https://arxiv.org/abs/2603.12201)  
**方向**: Efficient LLM / 稀疏注意力

**核心创新**:
DeepSeek Sparse Attention (DSA) 通过 lightweight lightning indexer 选择每 query 的 top-k 相关 token，将核心 attention 复杂度从 O(L²) 降至 O(Lk)。但 indexer 本身仍为 O(L²) 且每层独立运行，尽管相邻层的 top-k 选择高度相似。

IndexCache 利用这一跨层冗余：
- 将层划分为少量 **Full layers**（运行自己的 indexer）
- 多数 **Shared layers** 直接复用最近 Full layer 的 top-k 索引
- **Training-free IndexCache**：贪心搜索选择保留 indexer 的层，无需权重更新
- **Training-aware IndexCache**：多层蒸馏损失训练 indexer 匹配其服务层的平均 attention 分布

**实验结果**:
在 30B DSA 模型上：
- 可移除 75% 的 indexer 计算，质量几乎无损失
- **1.82× 预填充加速**，**1.48× decode 加速**
- 在 GLM-5 生产级模型上验证有效

### 3. Examining Reasoning LLMs-as-Judges in Non-Verifiable LLM Post-Training
**作者**: Yixin Liu 等  
**链接**: [arXiv:2603.12246](https://arxiv.org/abs/2603.12246)  
**方向**: Reasoning / LLM-as-Judge / RL Alignment

**核心创新**:
推理模型作为评判者（reasoning judges）可通过推理时扩展提升非可验证领域的评估质量。但在实际的 RL 对齐训练中，其有效性尚未被系统研究。

本文通过 controlled synthetic setting 发现：
- **非推理评判者**：容易导致 reward hacking
- **推理评判者**：可训练出在 gold-standard 评判下表现强的策略
- **意外发现**：推理评判者训练的策略学会生成**高效的对抗输出**，能在 Arena-Hard 等流行基准上欺骗其他 LLM 评判者

**研究启示**:
推理评判者在非可验证领域后训练中有潜力，但需要谨慎评估其训练出的策略可能利用评判者漏洞。

### 4. Neural Thickets: Diverse Task Experts Are Dense Around Pretrained Weights
**作者**: Yulu Gan 等 (MIT, NTU)  
**链接**: [arXiv:2603.12228](https://arxiv.org/abs/2603.12228)  
**代码**: https://github.com/sunrainyg/RandOpt  
**方向**: Efficient LLM / Post-training

**核心创新**:
预训练得到的参数向量通常被视为进一步迭代的起点。本文提出将其视为参数向量的分布，其 support 已包含 task-specific experts。

关键发现：
- **小模型**：专家解仅占分布体积的极小比例，需结构化优化（如梯度下降）才能发现
- **大模型**：任务专家密度显著增加，多样化的任务改进专家占据预训练权重邻域的相当比例

基于此，作者提出极简的并行后训练方法：
- 随机采样 N 个参数扰动
- 选择 top-K
- 通过多数投票集成预测

**实验结果**:
尽管方法简单，但与 PPO、GRPO、ES 等标准后训练方法 competitive。

### 5. Can RL Improve Generalization of LLM Agents? An Empirical Study
**作者**: Zhiheng Xi, Xin Guo, Jiaqi Liu 等 (复旦大学, 上海AI Lab)  
**链接**: [arXiv:2603.12011](https://arxiv.org/abs/2603.12011)  
**方向**: Agentic RL / Generalization

**核心创新**:
现有 RFT 评估大多在 in-domain 进行（训练测试同一环境）。本文系统研究 RFT 在环境变化下的泛化能力，考察三个维度：
- **环境内泛化**：跨任务难度
- **跨环境迁移**：未见环境（不同背景知识、观察空间、动作接口）
- **顺序多环境训练**：量化迁移与遗忘

**核心发现**:
- RFT 在同一环境内跨任务难度泛化良好
- 但向未见环境的迁移较弱，与语义先验和观察/动作接口的变化相关
- **顺序训练**在最小上游遗忘的情况下产生有前景的下游增益
- **混合训练**可改善整体平衡

### 6. On Information Self-Locking in Reinforcement Learning for Active Reasoning of LLM agents
**作者**: Deyu Zou 等 (港大, 阿里达摩院)  
**链接**: [arXiv:2603.12109](https://arxiv.org/abs/2603.12109)  
**方向**: Agentic RL / Active Reasoning

**核心创新**:
在主动推理（需要策略性地提问获取任务相关信息）中，用 RL 训练的 LLM agent 常遭遇 **information self-locking**：停止提出有信息量的问题，且难以内化已获得的信息。

机制分析：
- 将主动推理分解为 **Action Selection (AS)** 和 **Belief Tracking (BT)**
- 两者能力缺陷会限制 RL 训练中的信息探索
- 探索不足反过来阻碍 AS 和 BT 改进，形成反馈循环，将 agent 锁定在低信息状态

**解决方案**:
通过注入易于获取的 **directional critiques** 重新分配学习信号，帮助 agent 逃离 self-locking。在 7 个数据集上实现最高 **60% 的改进**。

### 7. XSkill: Continual Learning from Experience and Skills in Multimodal Agents
**作者**: Guanyu Jiang 等 (南京大学, 上海AI Lab)  
**链接**: [arXiv:2603.12056](https://arxiv.org/abs/2603.12056)  
**方向**: Agentic AI / Continual Learning / Multimodal

**核心创新**:
多模态 agent 在开放场景中面临工具使用效率低、编排不灵活的问题。XSkill 提出**双通道框架**，通过两种互补的可复用知识实现无需参数更新的持续改进：

- **Experiences**：提供动作级指导，用于工具选择和决策
- **Skills**：提供结构化任务级指导，用于规划和工具使用

**技术特点**:
- 积累和推理均基于 **visual observations**
- 通过视觉 grounded summarization 和 cross-rollout critique 提炼知识
- 推理时根据当前视觉上下文检索和适配知识
- 使用历史反馈形成持续学习闭环

**实验结果**:
在 5 个跨领域 benchmark 上，XSkill 持续大幅超越 tool-only 和 learning-based baselines，两种知识流在影响 agent 推理行为方面发挥互补作用。

### 8. Temporal Straightening for Latent Planning
**作者**: Ying Wang 等 (伊利诺伊大学, 密歇根大学)  
**链接**: [arXiv:2603.12231](https://arxiv.org/abs/2603.12231)  
**方向**: World Models / Latent Planning

**核心创新**:
预训练视觉编码器产生强语义特征，但并非为规划定制，包含对规划无关甚至有害的信息。受人类视觉处理的 **perceptual straightening** 假说启发，本文提出 temporal straightening 改进 latent planning 的表示学习。

方法：
- 使用 **curvature regularizer** 鼓励局部 straightened 的 latent 轨迹
- 联合学习 encoder 和 predictor
- 减少曲率使欧氏距离更好地近似 geodesic 距离
- 改善规划目标的 conditioning

**实验结果**:
Temporal straightening 使基于梯度的规划更稳定，在一系列目标达成任务上显著提升成功率。
