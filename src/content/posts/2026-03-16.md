### 1. IndexCache: Accelerating Sparse Attention via Cross-Layer Index Reuse
**作者**: Yushi Bai, Qian Dong, Ting Jiang, Xin Lv, Zhengxiao Du 等 (清华/智谱AI)
**链接**: [arXiv:2603.12201](https://arxiv.org/abs/2603.12201) | [代码](https://github.com/THUDM/IndexCache)
**方向**: 高效 LLM / 稀疏注意力

**核心创新**:
IndexCache 针对 DeepSeek Sparse Attention (DSA) 的瓶颈——轻量级索引器仍需 O(L²) 计算且每层独立运行——提出了跨层索引复用机制。核心观察是：相邻层的 top-k 索引选择高度相似。

技术方案包括：
1. **分层架构**: 将层分为 Full 层（运行独立索引器）和 Shared 层（复用最近 Full 层的索引）
2. **Training-Free 配置**: 贪心搜索算法在校准集上最小化语言建模损失，无需权重更新
3. **Training-Aware 蒸馏**: 多层级蒸馏损失训练保留的索引器，使其服务于多层注意力分布的平均

**实验结果**:
在 30B DSA 模型上，IndexCache 可移除 75% 的索引器计算，预填充加速 1.82×，解码加速 1.48×，质量损失可忽略。GLM-5 生产级模型初步实验验证了该方法的普适性。

### 2. Automatic Generation of High-Performance RL Environments

### 3. Spatial-TTT: Streaming Visual-based Spatial Intelligence with Test-Time Training
**作者**: Fangfu Liu, Diankun Wu, Jiawei Chi 等 (清华/腾讯)
**链接**: [项目页](https://liuff19.github.io/Spatial-TTT/) | [代码](https://github.com/THU-SI/Spatial-TTT)
**方向**: 具身智能 / 视觉基础模型

**核心创新**:
人类通过视觉流感知空间，核心挑战不是更长上下文窗口，而是空间信息的选择、组织和保留。Spatial-TTT 通过测试时训练 (TTT) 使模型适应流式空间证据：

1. **混合架构**: 大 chunk 更新与滑动窗口注意力并行，实现高效空间视频处理
2. **空间预测机制**: 应用于 TTT 层的 3D 时空卷积，鼓励模型捕捉跨帧几何对应和时间连续性
3. **密集 3D 空间描述数据集**: 引导模型以结构化方式记忆和组织全局 3D 空间信号

**实验结果**:
Spatial-TTT 在长程空间理解上取得 SOTA，在视频空间基准上表现优异。

### 4. EVATok: Adaptive Length Video Tokenization for Efficient Visual Autoregressive Generation
**作者**: Tianwei Xiong, Jun Hao Liew, Zilong Huang 等 (香港大学)
**链接**: [项目页](https://silentview.github.io/EVATok/) | [代码](https://github.com/HKU-MMLab/EVATok)
**方向**: 视觉生成 / 视频 Tokenization

**核心创新**:
传统视频 tokenizer 对所有视频片段采用统一 token 分配，在简单静态片段上浪费 token，而在复杂动态片段上 token 不足。EVATok 提出自适应 token 分配：

1. **最优 token 分配估计**: 为每个视频计算质量-成本权衡最优的 token 分配
2. **轻量级路由器**: 快速预测最优分配
3. **自适应 tokenizer**: 根据路由器预测的分配进行视频编码

**实验结果**:
在 UCF-101 类到视频生成上达到 SOTA，相比 LARP 和固定长度基线平均节省 24.4% token 使用量。

**方向**: 参数空间分析 / 模型适应

**核心创新**:
预训练产生学习到的参数向量，传统上被视为进一步迭代适应的起点。本文将预训练结果视为参数向量分布，其支持已包含任务特定专家：

1. **小模型**: 任务专家解占据分布体积的可忽略部分，发现依赖梯度下降等结构化优化方法
2. **大模型**: 任务专家密度急剧增加，多样化、任务改进的专家在预训练权重邻域中占据相当大比例

**方法**: 简单的全并行后训练——随机采样 N 个参数扰动，选择 top K，通过多数投票集成预测——与 PPO、GRPO、ES 等标准后训练方法具有竞争力。

### 6. EndoCoT: Scaling Endogenous Chain-of-Thought Reasoning in Diffusion Models
**作者**: Xuanlang Dai, Yujie Zhou, Long Xing 等 (上海AI Lab/InternLM)
**链接**: [项目页](https://internlm.github.io/EndoCoT/) | [代码](https://github.com/InternLM/EndoCoT)
**方向**: 推理 / 扩散模型

**核心创新**:
MLLM 作为文本编码器集成到扩散框架中存在两个关键限制：
1. 单步编码无法激活 Chain-of-Thought 过程
2. 解码过程中指导保持不变，阻止 DiT 将复杂指令逐步分解为可执行去噪步骤

EndoCoT 解决方案：
1. **迭代思维引导模块**: 激活 MLLM 推理潜力，通过迭代细化潜在思维状态
2. **终端思维接地模块**: 确保推理轨迹保持基于文本监督的接地，将最终状态与真实答案对齐

**实验结果**:
在 Maze、TSP、VSP、Sudoku 等基准上平均准确率 92.1%，超越最强基线 8.3 个百分点。

### 7. Budget-Aware Value Tree Search for LLM Agents
**作者**: Yushu Li, Wenlong Deng, Jiajin Li, Xiaoxiao Li
**链接**: [arXiv:2603.12634](https://arxiv.org/abs/2603.12634)
**方向**: Agentic AI / 推理搜索

**核心创新**:
当前测试时扩展方法将计算视为无限资源，导致 token 和工具预算浪费。BAVT（Budget-Aware Value Tree）是无需训练的推理时框架：

1. **步骤级价值估计**: 将多跳推理建模为动态搜索树
2. **预算条件节点选择**: 使用剩余资源比例作为节点价值的自然缩放指数，在预算耗尽时从广泛探索平滑过渡到贪婪利用
3. **残差价值预测器**: 评分相对进度而非绝对状态质量，缓解 LLM 自评估过度自信问题

**理论保证**: BAVT 在有限预算下以至少 1-ε 概率达到终止答案。

**实验结果**:
在四个多跳 QA 基准上，BAVT 在低预算约束下超越 4× 资源分配的基线性能。

### 8. From Sparse to Dense: Multi-View GRPO for Flow Models via Augmented Condition Space
**作者**: Jiazi Bu, Pengyang Ling, Yujie Zhou, Yibin Wang 等 (上海AI Lab/港中文)
**链接**: [项目页](https://bujiazi.github.io/mvgrpo.github.io/)
**方向**: RLHF / 文生图

**核心创新**:
标准 GRPO 中，将生成样本组与单一条件评估存在样本间关系探索不足的问题。MV-GRPO（Multi-View GRPO）通过增强条件空间创建密集多视图奖励映射：

1. **条件增强器**: 为同一提示生成的样本组生成语义相邻但多样化的标题
2. **多视图优势重估计**: 捕捉多样化语义属性，提供更丰富的优化信号
3. **概率分布推导**: 将原始样本基于新标题的条件概率纳入训练，无需昂贵样本再生

**实验结果**:
MV-GRPO 在文本到图像流模型对齐上超越 SOTA 方法。
