## 论文详解

### 1. When Right Meets Wrong: Bilateral Context Conditioning with Reward-Confidence Correction for GRPO
**作者**: Yu Li, Tian Lan, Zhengling Qi 等  
**链接**: [arXiv:2603.13134](https://arxiv.org/abs/2603.13134) | [GitHub](https://github.com/Skylanding/BiCC)  
**方向**: RL Training / Reasoning

**核心创新**:  
GRPO虽然基于组内均值计算优势，但在优化时把每个输出当作独立样本，忽略了同一组内正确与错误解之间的对比信号。本文提出对比式GRPO重构，证明GRPO目标隐式最大化正确/错误样本的策略比率边界。基于此提出 **Bilateral Context Conditioning (BICC)**，允许模型在优化时交叉引用成功和失败的推理轨迹，实现跨样本的直接信息流。进一步提出 **Reward-Confidence Correction (RCC)**，利用奖励-置信度协方差动态调整GRPO的优势基线，稳定训练。两种机制无需额外采样或辅助模型，可适配所有GRPO变体。

**实验结果**:  
在数学推理基准上，BICC-RCC在多个模型和算法上均取得一致改进。代码已开源。

### 2. ToolTree: Efficient LLM Agent Tool Planning via Dual-Feedback Monte Carlo Tree Search and Bidirectional Pruning
**作者**: Shuo Yang, Soyeon Caren Han, Yihao Ding, Shuhe Wang, Eduard Hoy 等  
**链接**: [arXiv:2603.12740](https://arxiv.org/abs/2603.12740)  
**方向**: Agentic AI / Tool Planning  
**会议**: ICLR 2026

**核心创新**:  
现有LLM Agent工具规划方法多采用贪婪、反应式策略，缺乏前瞻性且忽略工具间依赖。ToolTree提出**双阶段LLM评估**和**双向剪枝机制**的MCTS规划范式：在工具执行前后剪枝低潜力分支，使Agent能在扩展工具使用序列中做出知情、自适应决策。双向剪枝机制包括前置剪枝（基于工具描述和历史）和后置剪枝（基于执行反馈），显著提升规划效率。

**实验结果**:  
在4个开放集和封闭集工具规划基准上，ToolTree相比SOTA规划范式平均提升约**10%**，同时保持最高效率。

### 3. Steve-Evolving: Open-World Embodied Self-Evolution via Fine-Grained Diagnosis and Dual-Track Knowledge Distillation
**作者**: Zhengwei Xie, Zhisheng Chen, Ziyan Weng, Tingyu Wu, Chenglong Li, Vireo Zhang, Kun Wang 等  
**链接**: [arXiv:2603.13131](https://arxiv.org/abs/2603.13131)  
**方向**: Embodied AI / Self-Evolution

**核心创新**:  
开放世界具身智能体的瓶颈不是单步规划质量，而是交互经验的组织与进化方式。Steve-Evolving提出**非参数化自进化框架**，将细粒度执行诊断与双轨知识蒸馏紧密耦合形成闭环。三阶段流程：**Experience Anchoring** 将子目标尝试固化为结构化经验元组；**Experience Distillation** 将成功轨迹泛化为可重用技能，失败轨迹蒸馏为可执行护栏；**Knowledge-Driven Closed-Loop Control** 将技能和护栏注入LLM规划器，诊断触发的局部重规划在线更新约束。整个过程无需任何模型参数更新。

**实验结果**:  
在Minecraft MCU长程任务套件上，相比静态检索基线取得一致改进。

### 4. Visual-ERM: Reward Modeling for Visual Equivalence
**作者**: Ziyu Liu, Shengyuan Ding, Xinyu Fang, Xuanlang Dai, Penghui Yang, Jianze Liang, Jiaqi Wang, Kai Chen, Dahua Lin, Yuhang Zang 等 (上海AI Lab / InternLM)  
**链接**: [arXiv:2603.13224](https://arxiv.org/abs/2603.13224) | [GitHub](https://github.com/InternLM/Visual-ERM)  
**方向**: Multimodal Reward Model

**核心创新**:  
视觉到代码任务要求模型将图表、表格、SVG等结构化视觉输入高保真重建为可执行表示。现有奖励信号要么依赖文本规则，要么使用粗粒度视觉嵌入相似度，均无法捕捉细粒度视觉差异且易受reward hacking。Visual-ERM是**多模态生成式奖励模型**，直接在渲染视觉空间提供细粒度、可解释、任务无关的反馈。集成到RL中，使Qwen3-VL-8B-Instruct在chart-to-code任务上提升**+8.4**，在表格和SVG解析上也取得一致增益。

**实验结果**:  
引入的VisualCritic-RewardBench基准测试中，Visual-ERM-8B显著优于Qwen3-VL-235B-Instruct，接近领先闭源模型。证明细粒度视觉奖励监督对vision-to-code RL既必要又充分。

### 5. PhysMoDPO: Physically-Plausible Humanoid Motion with Preference Optimization
**作者**: Yangsong Zhang, Anujith Muraleedharan, Rikhat Akizhanov, Abdul Ahad Butt, Gül Varol, Pascal Fua, Fabio Pizzati, Ivan Laptev 等  
**链接**: [arXiv:2603.13228](https://arxiv.org/abs/2603.13228)  
**方向**: Humanoid Robotics / Motion Generation

**核心创新**:  
文本条件人体动作生成的最新进展主要由扩散模型驱动。现有方法通过全身控制器(WBC)将扩散生成动作转换为可执行轨迹，但WBC轨迹虽符合物理却可能与原始动作大幅偏离。PhysMoDPO是**直接偏好优化框架**，将WBC集成到训练流程中，优化扩散模型使WBC输出既符合物理又忠实于原始文本指令。使用物理和任务特定奖励为合成轨迹分配偏好。

**实验结果**:  
在文本到动作和空间控制任务上，PhysMoDPO在物理真实感和任务指标上均取得一致改进。在模拟和真实世界G1人形机器人上的零样本动作迁移也取得显著改进。

### 6. Video Streaming Thinking: VideoLLMs Can Watch and Think Simultaneously
**作者**: Yiran Guan, Liang Yin, Dingkang Liang, Jianzhong Ju, Zhenbo Luo, Jian Luan, Yuliang Liu, Xiang Bai 等 (小米)  
**链接**: [arXiv:2603.12262](https://arxiv.org/abs/2603.12262) | [GitHub](https://github.com/1ranGuan/VST)  
**方向**: Streaming Video Understanding

**核心创新**:  
在线VideoLLM对支持响应式实时交互至关重要。现有方法专注流式感知，缺乏同步逻辑推理流；直接应用test-time scaling会引入不可接受的延迟。VST提出**边看边思考**机制，在视频流接收期间激活推理，通过将LLM推理延迟摊销到视频播放中，实现及时理解、连贯认知与实时响应性的平衡。提出完整后训练流程：VST-SFT将离线VideoLLM结构适配为因果流式推理，VST-RL在多轮视频交互环境中通过自探索提供端到端改进。

**实验结果**:  
VST-7B在在线基准上表现强劲：StreamingBench **79.5%**，OVO-Bench **59.3%**。相比Video-R1，VST响应速度快**15.7倍**，在VideoHolmes上提升**+5.4%**。

### 7. Efficient and Interpretable Multi-Agent LLM Routing via Ant Colony Optimization
**作者**: Xudong Wang, Chaoning Zhang, Jiaquan Zhang, Chenghao Li, Qigan Sun, Sung-Ho Bae, Peng Wang, Ning Xie, Jie Zou, Yang Yang, Hengtao Shen 等  
**链接**: [arXiv:2603.12933](https://arxiv.org/abs/2603.12933)  
**方向**: Multi-Agent Systems / Efficient LLM

**核心创新**:  
LLM驱动的多智能体系统(MAS)在复杂推理和工具使用上展现强大能力，异构智能体池进一步扩展了质量-成本权衡空间。但高推理成本、延迟和有限透明度限制了实际部署。AMRO-S提出**基于蚁群优化的高效可解释路由框架**：1) 使用监督微调的小型语言模型进行意图推断；2) 将路由记忆分解为任务特定信息素专家，减少任务间干扰；3) 质量门控异步更新机制将推理与学习解耦。通过结构化信息素模式提供可追踪的路由证据。

**实验结果**:  
在5个公开基准和高并发压力测试中，AMRO-S在质量-成本权衡上持续优于强路由基线。

### 8. CRYSTAL Benchmark for Transparent Multimodal Reasoning Evaluation
**作者**: Wayner Barrios, SouYoung Jin 等  
**链接**: [arXiv:2603.13099](https://arxiv.org/abs/2603.13099)  
**方向**: Multimodal Reasoning / Evaluation

**核心创新**:  
CRYSTAL (Clear Reasoning via Yielded Steps, Traceability, and Logic) 是通过可验证中间步骤评估多模态推理的诊断基准，包含**6,372**个实例。提出两个互补指标：**Match F1** 通过语义相似度匹配评分步骤级精确率和召回率；**Ordered Match F1** 进一步惩罚无序推理链。参考通过Delphi启发式流程构建：四个独立MLLM生成轨迹，经语义聚类和人工质量门验证。此外提出**Causal Process Reward (CPR)**，将答案正确性与步骤级对齐耦合的乘法奖励，以及逐步增加推理难度的CPR-Curriculum。

**实验结果**:  
对20个MLLM的评估显示：没有竞争模型能保持超过**60%**的匹配步骤在正确顺序中。CPR-Curriculum通过GRPO在Match F1上取得**32%**改进，而加性奖励策略失败。
